# java-learning
Java Loop Programming | Fundamental DSA Concepts Collection of loop-based programs: ForLoop, DoWhileLoop, Prime Number Checker, Factorial Calculator, Number Reversal. Building strong programming foundation for BTech CSE.







# Java Loops Learning 🚀

Day 1 of my Java DSA journey before college starts.

## Programs Included

### 1. **Forloop.java**
- Multiplication table generator
- Uses for loop to iterate and print




### 2. **Dowhileloop.java**
- Number input program
- Do-while loop execution
- Runs at least once before condition check




### 3. **Reversesnumber.java**
- Reverses given number
- Uses while loop
- Example: 12345 → 54321




### 4. **Primeornot.java**
- Checks if number is prime
- Loop-based iteration
- Returns true/false





### 5. **Factorialnumber.java**
- Calculates factorial of number
- For loop with multiplication
- Example: 5! = 120





## Concepts Covered
- While loops
- Do-while loops
- For loops
- Nested loops
- Loop control
