# Java Learning Repository

A structured collection of **32 Java programs** covering fundamentals, loops, patterns, methods, and problem-solving logic. Built as a pre-college foundation for competitive programming and interviews.

---

## 📁 What's Included

### Loops (5 programs)
Core loop concepts - ForLoop, DoWhileLoop, PrimeOrNot, FactorialNumber, ReverseOfNumber

### Patterns (4 programs)
Nested loop patterns - CharacterPattern, HalfPyramid, InvertedStar, NestedLoops

### Part-13 (13 programs)
Methods, functions, and logic problems - BinarytoDecimal, Binomial, Palindrome, PrimeRange, SumofDigits, Swap, MethodOverloading, and more

### Pattern-2 (9 programs)
Advanced patterns - Butterfly, Diamond, Floyd, HollowRectangle, HollowRhombus, Rhombus, BinaryTriangle, Expo, InvertedRotatedPyramid

### Pattern-1 (1 program)
Standalone pattern file
